# 🐾 Wildvine — Animal Charity Website

**"Give Love for Saving World Animals."**

A fully responsive charity website for an animal welfare organization, showcasing its mission, services, and ways to donate — built with **HTML5, CSS3, and JavaScript**.

Built as part of the **Crixsoft Solution Web Development Internship** 🚀

## 📸 Preview

![Wildvine Preview](./readme-images/desktop.png)

## ✨ Features

- 🏠 **Hero Section** — mission-driven headline with a prominent donation call-to-action
- ℹ️ **About Section** — organization story and impact banners
- 🐕 **Services Section** — overview of animal welfare services with a location/service map
- ❤️ **Donate Section** — highlighted donation cards to support different causes
- 📅 **Event Section** — upcoming charity events and campaigns
- 🤝 **Partners Section** — logos of partner organizations with hover states
- 💬 **Testimonials** — supporter feedback carousel
- 📷 **Instagram Feed Preview** — gallery-style social media section
- 🌐 **Multi-language Selector** — English / French / Spanish toggle in the header
- 📱 **Fully Responsive Navigation** — mobile-friendly slide-out menu

## 🛠️ Built With

- **HTML5** — semantic, accessible markup
- **CSS3** — responsive layout with flexbox/grid, custom properties
- **JavaScript (Vanilla)** — mobile nav toggle, search toggle, and UI interactivity
- **Ionicons** — icon set used throughout the site

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)
- [Git](https://git-scm.com/downloads) installed on your machine

### Run Locally

```bash
# Clone the repository
git clone https://github.com/<abrarquadri>/Crixsoft-Solution_Wildvine-Charity-Website.git

# Navigate into the project folder
cd Crixsoft-Solution_Wildvine-Charity-Website

# Open index.html in your browser
```

No build steps or dependencies required — it's pure HTML, CSS, and JS.

## 📱 Responsive Design

The layout is optimized for desktop, tablet, and mobile, with a collapsible slide-in navigation menu and reflowing content grids across the About, Services, Donate, and Partners sections.
